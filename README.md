# kadai1
ラズベリーパイを用いてデバイスドライバを作りました。これはそのデバイスドライバを用いて取り組んだ課題1のリポジトリです。

# 内容
LEDをGPIO25とGNDにつなぎ、ターミナルにて`echo 1 > /dev/myled0`と入力するとLEDが点灯、`echo 0 > /dev/myled0`と入力すると消灯するプログラムです。
